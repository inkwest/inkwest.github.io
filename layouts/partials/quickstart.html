Quick Start

terminal 
# Clone the repository
$ git clone {{ .Site.Params.repoURL }}.git
$ cd inkwest

# Configure environment
$ cp .env.example .env
$ # Edit .env with your AI provider API key

# Start all services
$ docker compose up -d

# Run database migrations
$ docker compose exec backend python db_migrate.py

# Open in browser
$ open http://localhost:3000
